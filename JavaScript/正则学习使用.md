###  正则的学习及使用

#### 元字符

|  元字符 | 描述                           |
| --  |  --|
|  .      | 查找单个字符，除了换行和行结束符 |
| \w |  查找单词字符 |
| \W |  查找非单词字符 |
| \d |  查找数字 |
| \D |  查看非数字字符 |
| \s |  查找空白字符 |
| \s |  查找空白字符 |
| \b |  查找单词边界 |
| \B |  查找非单词边界 |
| \0 |  查找Null字符 |
| \n | 查找换行符|
| \f  | 查找换页符|
| \r | 查找回车符|
| \t | 查找换页符|
| \v | 查找垂直制表符|
| \xxx | 查找以八进制 ***规定的字符|
| \xdd | 查找以十六进制 dd规定的字符|
| \uxxx | 查找以十六进制 ****规定的UNIcode字符
